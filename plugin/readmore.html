Chrome Carbon Tracker

The Chrome Carbon Tracker is a chrome extension that approximates the carbon equivalents emitted during a browsing session.
It does so by checking the amount of data which the browser receives and connecting the IP adress of the sender to a given country. Knowing which country the server is in lets us approximate how carbon intensive the energy it uses is.

To calculate the energy usage we use the ratio 0.81 kWh/GB which is the same as is used on sustainablewebdesign.org.
This number was chosen since there is no consensus on one "true" number for this, looking at "Electricity Intensity of Internet DataTransmission" by J. Alsan and K. Mayers from 2015, we can see that the energy consumption pr. data unit varies up to five orders of magnitude.
Lastly we connect to electricitymaps api get the real time value for gCO2equiv/kWh in the country where the server is located.
